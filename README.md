***
## 프로그램 개요
1. 각 요일에 추천할 카테고리를 무작위로 정함
2. 각 코치가 해당 요일에 먹을 메뉴를 추천한다.

# 📜 기능 명세서 📜

***

### 입출력
- [ ] 메뉴 추천받을 코치의 이름 입력받기
- [ ] 각 코치가 못먹는 메뉴 입력받기

### 주요 기능
- [ ]

# ❗️ 예외 상황 ❗

***
### 공통 예외
- [ ] **공백 입력** ex) ` `

### 입력시 예외
- [ ] 코치의 이름이 2글자 ~ 4글자 인지 
- [ ] 한번에 최소 2명 ~ 최대 5명까지만 식사 가능
- [ ] 못먹는 메뉴가 0개~2개가 있다.
- [ ] 한 주에 같은 카테고리는 최대 2회까지만 가능
- [ ] 못먹는 메뉴가 메뉴에 존재하지 않는 경우
